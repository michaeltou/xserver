URL地址为：http://web.yunmo.cn/asmx/smsservice.aspx

1. 参数定义

序号 参数 说明
1 name 必填参数。用户账号
2 pwd 必填参数。（web平台：基本资料中的接口密码）
3 content 必填参数。模板内容 UTF-8编码
格式：固定内容@固定内容@固定内容
@ 可代表任意多个字符；可变的地方用@代替
例子：您的验证码为：@ 20分钟内有效。您获得了@个积分，验证后在个人中心领取【企业签名】
4 tid 添加时不填
修改时填模板的ID号；
如id号不存在则返回  3,tid is not true
5 type 必填参数。固定值 operate_templet


2. 提交响应

用户短信通过http请求提交到服务器后，服务器返回响应码，响应码的格式如下：
code,tid
3. 格式说明

提交响应为英文逗号隔开的一行数据，状态,模板ID号
如果响应的状态不是“0”，状态,消息。


3.1 响应状态值说明

代码 说明
0 提交成功
1 含有敏感词汇
2 余额不足
3 tid is not true
10 账号不存在
11 账号注销
12 账号停用
13 IP鉴权失败
14 格式错误
-1 系统异常
 
3.2 示例

http://web.yunmo.cn/asmx/smsservice.aspx?name=test& pwd=11dfgsdfsdfsdf2345&content=您的验证码为@，20分钟内有效【企业签名】 &type=pt_audio
