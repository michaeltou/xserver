1.短信办理

1.1 短信使用时间有期限吗？
 
  没有期限。长期有效，不会过期！

1.2 你们可以开发票吗？
 
  可以开票，普通发票和增值税发票都可以开。可自助申请。

2.短信使用

2.1 可以同时发多个手机吗？
 
  可以，相同内容一次最多提交1000个号码。
 
2.2 三网（移动、联通和电信）都可以发吗？
 
  是的。三网都支持，全国号码都可以发。
 

3.短信签名

3.1 什么是短信签名？
 
  短信签名是指代表公司/产品的标示性文字，一般位于短信内容【】中！

3.2 短信签名规范？

  2-8个字符;不能是纯数字、纯字母、数字和字母的组合，不含违禁关键词，不能有其他特殊符号。

3.3 短信签名报备所需材料？

  企业营业执照、组织机构代码证、税务登记证。

3.4 验证码模板和通知模板如何调用签名？

  验证码模板和系统通知模板，可以调用任意的系统签名；

3.5 短信签名个数限制？

  每位用户可提供至少5个固定签名。

3.6 短信签名审核和生效时间？

  短信签名提交审核后3个工作日内完成审核，审核通过后立即生效。

3.7 短信签名为何要审核，可不可以不写？

  为配合三大运营商的业务要求，同时也可以侧面保证品牌效应。短信在发送时，需签署已经备案的签名，为必填！

4.短信模板

4.1 短信模板的审核周期？

  2个工作日内

4.2 短信模板怎么设置？

  提供7个固定模版；企业用户可上传自定义模版（不支持营销短信），不限个数；个人用户不支持自定义模版。不支持全变量，需人工审核。 
  比如，模板内容：云模手机动态验证码：@，如非本人操作，请忽略。“签名”部分不是模板的内容，请勿添加至模板内容中。调用接口的时候，需传“签名”、“模板变量”！如：签名为“云模云服务”，模板变量为“8888”，最后的短信内容为：“云模手机动态验证码：8888，如非本人操作，请忽略。【云模云服务】”

5.短信计价方式

5.1 短信内容计算规则？

  模板文案内容字数 + 签名字数 = 短信字数 短信字数<=70个字数，按照70个字数一条短信计算 短信字数>70个字数，即为长短信，按照67个字数记为一条短信计算

5.2 短信的计费模式？

5.2.1 按照实际使用情况进行实时扣费；运营商返回短信发送成功状态才会扣费；若短信发送5分钟后还未返回状态，将先扣费，实际返回状态后根据失败条数进行短信量返补。 

5.2.2 按阶梯计费，用的越多单价越低；月消费金额＝月累计短信成功发送量所在的档位＊月累计短信发送成功量； 

  比如，模板内容：您的验证码是@，非本人操作请忽略！ “签名”部分不是模板的内容，请勿添加至模板内容中。调用接口的时候，需传“签名”、“模板变量”

5.3 短信发送失败，还是会扣费吗？

  接口调用失败及运营商返回失败状态，都不会扣费
