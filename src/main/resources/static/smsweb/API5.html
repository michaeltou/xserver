1. 接口地址

http://web.yunmo.cn/asmx/smsservice.aspx

2. 参数定义

 
序号 参数 说明
1 name 必填参数。用户账号
2 pwd 必填参数。用户密码
3 type 必填参数。固定值 balance


3. 提交响应


 
code为获取状态，balance是用户余额。


4. 提交响应值

代码 说明
0 成功
10 账号不存在
11 账号注销
12 账号停用
13 IP鉴权失败
 

5. 示例

http://web.yunmo.cn/asmx/smsservice.aspx?name=test&pwd=testpwd&type=balance
